A client-server TCP memory game:
made for two different users on a network.
creates a matrix of n*n cards, on each card there is a picture, each picture appears twice.
the card are flipped(picture is hidden), each turn a player flips two card. 
if the cards have the same picture the cards remain revealed and the user gets a point, if not the cards are flipped back after 3 seconds.
both users see the board live.
the game ends when all the cards are reaveald.

there could be multiple games at once.
for each two clients that connect the sever, a game is starting.

* i have no rights on the pictures.
